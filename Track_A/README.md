- **OS**: Windows 11, Version 24H2, Build 26100.7171
- **CPU**: 12th Gen Intel(R) Core(TM) i5-12600KF

## How to Run
1. Open Command Prompt (CMD)
2. Navigate to `Track_A\scripts` directory
3. Run `run_experiments.bat`

## Output Locations
- **Plots**: Generated in `scripts` directory
- **Raw Data**: Saved in `scripts\raw_data` directory
- **Executable**: Built in `scripts\build` directory
